## 3. 数据集同步与本地产物

### 3.1 数据集同步脚本（全量优先）

- 脚本：`scripts/build_dataset_catalog.py`
- 命令：
  - `npm run sync:datasets`（全量优先）
  - `npm run sync:datasets:preview`（仅预览模式）
- 下载策略（开源数据集）：
  1. HuggingFace datasets-server parquet 接口
  2. `datasets` 库 fallback（`save_to_disk` 产出 `raw/hf`）
  3. GitHub 原始文件 fallback
  4. 最后 fallback sample（避免前端空白）
- 支持 token：`HF_TOKEN` / `HUGGINGFACE_TOKEN`
- HuggingFace 缓存目录固定到项目内 `.cache/huggingface`

### 3.2 已落地产物

- `public/datasets/catalog.json`
- `public/datasets/open-source/<id>/sample.jsonl`
- `public/datasets/open-source/<id>/raw/parquet/*`（可用时）
- `public/datasets/open-source/<id>/raw/hf/*`（datasets fallback）
- `public/datasets/open-source/<id>/raw/source/*`（github fallback）
- `public/datasets/self-built/<id>/sample.jsonl`

### 3.3 ImageNet 特殊处理

- `imagenet` 使用 `ILSVRC/imagenet-1k`
- 支持“单个 parquet 分片”下载路径（避免一次拉完整快照）
- 若 token 未开启 gated 权限会出现 403（需在 HF token 权限中放开 public gated repo 访问）

## 4. Parquet/Arrow 转换为 test.jsonl

### 4.1 转换脚本

- 脚本：`scripts/parquet_to_test_jsonl.py`
- 命令：`npm run parquet:to-test-jsonl`

### 4.2 支持输入

- `raw/parquet/*.parquet`
- `raw/hf/`（HuggingFace `save_to_disk`，底层 arrow）

### 4.3 输出规则

- 每个数据集目录输出：`<dataset>/test.jsonl`
- 若包含图片字段（bytes 或 path）：
  - 写入 `<dataset>/image/`
  - 在 `test.jsonl` 中存相对路径（如 `image/image_00000001.jpg`）

## 7. 已知限制

1. 配置文件写入目前仍是 Vite 本地中间件方案（开发期），未接真实后端服务。
2. 部分 gated 数据集（典型是 ImageNet）依赖 HF 账号权限与 token 配置，可能失败。
3. 任务执行、推理、评测、报告回传主链路尚未接入（当前重点是“建任务 + 配置落盘 + 数据集可视化”）。

## 11. 测试用例全流程实现（6大类30项）

### 修改目的
根据《网安-加速卡测试用例》文档，将加速卡评测模块从简单的 OpenCompass 精度评测扩展为覆盖6大测试类别、30个测试项的完整测试平台。

### 6大测试类别
1. **芯片基础测试** — 通信带宽、显存带宽、算力、功耗、稳定性
2. **模型训练性能测试** — PyTorch DDP 单/多卡、Megatron 分布式
3. **模型推理性能测试** — vLLM benchmark、TensorRT-LLM
4. **模型精度测试** — OpenCompass、EvalScope
5. **生态兼容性测试** — Verl、RAG、CUDA 迁移、GPU 虚拟化
6. **视频编解码测试** — MacCodec、Heepstream、FFmpeg

### 新增文件（后端）
| 文件 | 说明 |
|------|------|
| `backend/app/schemas/test_case.py` | 完整数据模型（549行，枚举/配置/结果/预定义目录） |

| `backend/app/services/test_runners/chip_basic_runner.py` | 芯片基础测试执行器 |
| `backend/app/services/test_runners/training_runner.py` | 训练性能测试执行器 |
| `backend/app/services/test_runners/inference_runner.py` | 推理性能测试执行器 |
| `backend/app/services/test_runners/accuracy_runner.py` | 精度测试执行器 |
